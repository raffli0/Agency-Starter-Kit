Astro lets us ship static, content-focused websites without shipping a heavy JavaScript
runtime to the browser. By default, components render to HTML and only hydrate where
interactivity is needed.

For agencies, that means faster pages, better Core Web Vitals, and simpler maintenance for
client teams.
